# World Flags and Capitals Quiz

## Prompt
Create an engaging quiz about world flags and capitals for learners of mixed skill levels. The quiz should test recognition of country flags, matching countries to their capitals, and basic geography awareness. Include a balanced mix of easy, medium, and challenging questions. Present each question clearly, use multiple-choice format where helpful, and provide an answer key at the end.

### Quiz Requirements
- Cover countries from every continent.
- Include both flag identification and capital city questions.
- Avoid overly obscure countries unless used as challenge questions.
- Keep wording simple and educational.
- Make the quiz suitable for classroom use, self-study, or a friendly competition.

### Example Question Types
- Which country has this flag?
- What is the capital of this country?
- Match the flag to the correct country.
- Match the country to its capital.

## Analysis
This quiz topic works well because flags and capitals are visual, memorable, and easy to turn into interactive questions. It supports geography learning in two complementary ways: visual recognition through flags and factual recall through capital cities.

### Learning Goals
- Improve country recognition.
- Reinforce knowledge of world capitals.
- Build confidence with global geography.
- Encourage pattern recognition between regions, flags, and political geography.

### Difficulty Design
- Easy questions should focus on widely known countries such as Japan, France, Canada, and Brazil.
- Medium questions can include countries with less familiar capitals or similar-looking flags.
- Hard questions should use countries with commonly confused capitals or visually similar flags.

### Best Format
A multiple-choice format is ideal because it keeps the quiz accessible and fast-paced. Adding image-based flag questions makes the quiz more engaging, while a final score summary makes it motivating.

### Recommended Tone
The quiz should feel friendly, educational, and slightly competitive. It should avoid being too academic or too obscure, so participants stay interested and can learn as they go.
